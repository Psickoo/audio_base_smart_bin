The Sound Classifier Trash Bin is an innovative, automated waste sorting system that classifies discarded materials based on their unique acoustic signatures upon impact. Powered by a TinyML model trained via Edge Impulse, the system intelligently distinguishes between materials like glass, metal, and plastic by analyzing the sound they make when dropped. The core logic and hardware control are driven by C++ firmware running on a microcontroller, which handles real-time audio processing, edge inferencing, and the precise actuation of sorting motors. Furthermore, the physical hardware—featuring a specialized acoustic impact chamber and an automated sorting mechanism—is custom-designed using Autodesk Fusion 360 to ensure optimal sound capture and seamless mechanical operation.
